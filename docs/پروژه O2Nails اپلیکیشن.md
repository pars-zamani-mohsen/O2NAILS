## 🚀 **نحوه Build و نصب:**

### **Build پروژه:**
```bash
# Build کامل
cd /var/www/html/O2NAILS && ./gradlew build

# ساخت APK Debug
cd /var/www/html/O2NAILS && ./gradlew assembleDebug

# ساخت APK Release
cd /var/www/html/O2NAILS && ./gradlew assembleRelease
```

### **نصب روی شبیه‌ساز:**
```bash
# نصب APK
adb -s emulator-5554 install ./app/build/outputs/apk/debug/app-debug.apk

# بررسی نصب
adb -s emulator-5554 shell pm list packages | grep o2nails

# حذف نسخه قبلی (در صورت نیاز)
adb -s emulator-5554 shell pm uninstall com.o2nails.v11
```

### **مسیر فایل‌های APK:**
- **Debug APK:** `./app/build/outputs/apk/debug/app-debug.apk`
- **Release APK:** `./app/build/outputs/apk/release/app-release-unsigned.apk`
